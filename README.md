ToDos 
this is arguably that one project that everyone has in thier portfolio, how can I miss this?, So here we go

A RESTful backend API built with Python and FastAPI for Adding,Modifying,Deleting And Above All Tracking a List Of Todos 

Tech Stack used in this projects:
Framework: FastAPI
Database: SQLite
ORM: SQLAlchemy
Data Validation: Pydantic

Features
->Full CRUD functionality (Create, Read, Update, Delete) for workout logs.
->Advanced data validation ensuring proper formatting for sets, reps, and weights.
->Modular database architecture using SQLAlchemy ORM.
->Auto-generated Swagger UI documentation.

How to install and setup this project 

Step1. Clone the repository:
   
   git clone ("https://github.com/MEERANSAHIB/Todos")
   cd Todos
Step2. Create and activate virtual environment
  python -m venv fastapienv
.\fastapienv\Scripts\activate
Step3. Install The dependies
  pip install -r requirements.txt
That's all you're good to go
